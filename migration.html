Problem Statement

Legacy SharePoint on-premises sites relied on Nintex workflows and manual approval processes, which were not compatible with SharePoint Online. Users experienced delays and limited visibility, and IT faced challenges maintaining these systems securely.

Solution

Redesigned the site in SharePoint Online with modern lists and libraries. Rebuilt workflows using Power Automate to replace Nintex workflows, leveraging native Microsoft connectors for security and maintainability.

  • Document Library with automated approval workflow
  • SharePoint List for tracking requests and statuses
  • Power Automate cloud flows for approvals, notifications, and updates
  • Integration with Outlook for user notifications

Tools & Connectors

  • SharePoint Online
  • Power Automate (Triggers & Approvals)
  • Outlook Connector
  • Office 365 Users Connector

Results

  • Eliminated reliance on third-party Nintex workflows
  • Reduced approval delays and manual errors
  • Enhanced visibility for managers and IT
  • Simplified long-term maintenance and support
